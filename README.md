#### [Reading Speed](./one-file-exercises/ReadingSpeed.java)
Estimates the amount of time it would take to read a text. Takes the following inputs from stream:<br/>
`String speed`: any one of the options 'slow', 'medium', and 'fast'. Incorrect output returns an explicit message.<br/>
`int textLength`: how many words are in the text<br/>
Returns the estimate in minutes and hours.<br/>

__Note:__ The reading speed options are based on research results and reflect reading proficiency for middle schoolers, high schoolers, and adults.<br/>
